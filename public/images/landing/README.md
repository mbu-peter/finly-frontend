# Landing Page Images

This directory contains images used on the Finly landing page.

## Current Images (Uploaded)

### Feature Showcase Images
These are the actual uploaded images being used in the hero section:

1. **market.png** - Crypto market interface showing live prices and charts
2. **trade.png** - P2P trading platform interface
3. **transaction.png** - Transaction history and analytics page

## Image Specifications

- **Format**: PNG
- **Style**: Landscape orientation, high-quality screenshots
- **Theme**: Dark interface matching the app design
- **Usage**: Featured in overlapping layout on landing page

## Landing Page Integration

Images are displayed in an overlapping layout with:

- **Main Feature**: `market.png` - Base layer with gradient border
- **Right Overlap**: `trade.png` - Smaller overlay on the right
- **Left Overlap**: `transaction.png` - Smaller overlay on the left

Each image includes:
- Gradient blur effects
- Descriptive captions
- Feature highlights
- Call-to-action messaging
